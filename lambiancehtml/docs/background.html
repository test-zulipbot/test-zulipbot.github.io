A how-to guide to get the most out of your new template.

Background

Ambiance supports a fullscreen background video or a fullscreen background slideshow. Whether you use video or not it's a good idea to always provide images for the slideshow in case video is not supported (tablets / mobile phones).

Slides

On the bottom of each and every html file define your slides the following way:

<ul id="slideshow" data-speed="4000">
        <li>
            <img src="/assets/img/slideshow/slide.jpg">
        </li>
        <li>
            <img src="/assets/img/slideshow/slide.jpg">
        </li>
        <li>
            <img src="/assets/img/slideshow/slide.jpg">
        </li>
    </ul>

Make sure to do this on every page because users can navigate directly to a sub page. If you don't the slideshow won't show up. In case you want to modify the slideshow speed do the following:

Open style.css and around line 1513 find the selector #slideshow li.visible and change the value of animation-duration: 9s to your desired value. In case you use less adjust this property in the file slideshow.less Next open tje javascript file app.js and around line 321 change the value of setTimeout(start, 7000);. Set it to to the amount of seconds you used in the css file minus two seconds. So in case you set the css to 6 seconsds, use 4 seconds in the javascript. This is because of the fade transitions that needs some time as well.

We've allready created a slideshow folder for you in the assets folder but it's really up to you where you place the images.

Video

On each and every body tag you open define the <body data-video="116244713"> tag and provide the id of your Vimeo or Youtube video. Make sure to do this on every page because users can navigate directly to a sub page. If you don't the video won't show up.

Your ID is the last segment of the url in case of Vimeo:

https://vimeo.com/47868543 -> ID = 47868543

In case of you tube it's the segment after the = sign.

https://www.youtube.com/watch?v=3TTswKT9J9g -> ID = 3TTswKT9J9g
