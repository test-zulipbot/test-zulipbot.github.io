A how-to guide to get the most out of your new template.

Css

The css is written with the help of less css. It's very easy to use. Have a look at their website for documentation. Of course you can always edit the style.css directly. It's well commented so you should be able to find your way easily.

Js

Ambiance makes use of several js files.

App.js

This files is responsible for the overall logic. It loads pages, handles routing and inits the background video or slideshow.

Page.js

This lib is responsible for handling routing. It works pretty simple. Check out the website in case you need more information.

Backgoundvideo.js

As the name suggests this file is responsible for displaying the fullscreen background video. It works with both YouTube and Vimeo.

Slidenavigation.js

This file is responsible for the paralax effect in the main navigation.

Jquery-2.1.4.js

Jquery is included. More info on this can be found at the jquery website.

Photoswipe.js

This lib is used for displaying image galleries. Check out the website in case you need more information.

Remodal.js

This lib is used for displaying modals. Check out the website in case you need more information.
