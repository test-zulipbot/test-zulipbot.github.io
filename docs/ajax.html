A how-to guide to get the most out of your new template.

Pages are loaded via ajax.

It's possible however for a viewer to navigate to a sub page directly. In order to make sure this works you always have to make sure that your javascript files are included at the bottom of each and every sub page.

<script type="text/javascript" src="http://localhost/ambiance/output_dev/assets/js/libs/history.js"></script>
    <script type="text/javascript" src="http://localhost/ambiance/output_dev/assets/js/libs/modernizr.js"></script>
    <script type="text/javascript" src="http://localhost/ambiance/output_dev/assets/js/libs/page.js"></script>
    <script type="text/javascript" src="http://localhost/ambiance/output_dev/assets/js/libs/jquery-2.1.4.js"></script>
    <script type="text/javascript" src="http://localhost/ambiance/output_dev/assets/js/libs/backgroundvideo/backgroundvideo.js"></script>
    <script type="text/javascript" src="http://localhost/ambiance/output_dev/assets/js/libs/photoswipe/photoswipe.js"></script>

Page linking.

A page is animated into the screen after it's loaded. You can tell Ambiance what transition to use by setting a data-transition property on the href that links to your next screen:

<a href="/" class="back" data-transition="slide-from-top">

L'ambiance will automatically add and -in and -out class to your chosen transition.

We predefined transitions. But you can also define your own transitions. Have a look at transitions.less or line 602 of style.css 

.slide-from-top {
  -webkit-transition: transform 0.8s cubic-bezier(0.4, 0.01, 0.165, 0.99);
  transition: transform 0.8s cubic-bezier(0.4, 0.01, 0.165, 0.99);
  -webkit-transform: translateY(0);
  -ms-transform: translateY(0);
  transform: translateY(0);
}
.slide-from-top-in {
  -webkit-transform: translateY(-100%);
  -ms-transform: translateY(-100%);
  transform: translateY(-100%);
}
.slide-from-top-out {
  -webkit-transform: translateY(100%);
  -ms-transform: translateY(100%);
  transform: translateY(100%);
}
.slide-to-top {
  -webkit-transition: transform 0.8s cubic-bezier(0.4, 0.01, 0.165, 0.99);
  transition: transform 0.8s cubic-bezier(0.4, 0.01, 0.165, 0.99);
  -webkit-transform: translateY(0);
  -ms-transform: translateY(0);
  transform: translateY(0);
}
.slide-to-top-in {
  -webkit-transform: translateY(100%);
  -ms-transform: translateY(100%);
  transform: translateY(100%);
}
.slide-to-top-out {
  -webkit-transform: translateY(-100%);
  -ms-transform: translateY(-100%);
  transform: translateY(-100%);
}
.slide-left {
  -webkit-transition: transform 0.8s cubic-bezier(0.4, 0.01, 0.165, 0.99);
  transition: transform 0.8s cubic-bezier(0.4, 0.01, 0.165, 0.99);
  -webkit-transform: translateX(0);
  -ms-transform: translateX(0);
  transform: translateX(0);
}
.slide-left-in {
  -webkit-transform: translateX(100%);
  -ms-transform: translateX(100%);
  transform: translateX(100%);
}
.slide-left-out {
  -webkit-transform: translateX(-100%);
  -ms-transform: translateX(-100%);
  transform: translateX(-100%);
