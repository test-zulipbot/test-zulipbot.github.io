Adding a page to the home navigation

Adding a new page to the navigation is easy. First create a new folder and give it the name of your page. Copy a html page in it (for example about.html) and modify it with your content.

Next open the general index.html file and look for the <ul id="navigation"> tag.

Add a new list item to it:

<li>
    <a href="your-file.html" data-transition="slide-to-top">
        <section>
            <h1>Menu</h1>
            <h6>Our delicious dishes</h6>
        </section>
        <footer>
            <h3 class="serif">Our kitchen</h3>
            <p>Dishes and drinks</p>
        </footer>
    </a>
</li>

And refer to your new folder name in the <a href="your-file.html" data-transition="slide-to-top">

That's it!
