Preloader & Logo

Adjusting the logo is pretty easy. Firstly in your index.html have a look at the following code:

<!-- The splash screen -->
<div id="splash">
    <div class="loader">
        <img class="splash-logo" src="assets/img/logo/logo.svg"/>
        <div class="line"></div>
    </div>
</div>
<!-- End of splash screen -->

As you can see the current logo is linked to assets/img/logo/logo.svg but you can easliy link it to another file of your choice.

A bit further on in the same index.html file adjust the logo in the header as well.

<header>
    <a href="http://localhost/ambiance/output_dev" class="logo">
        <h1>Home</h1>
        <img src="http://localhost/ambiance/output_dev/assets/img/logo/logo.svg" alt="Home" />
    </a>
</header>

Important

Make sure to make these adjustments on each and every page! Ie: about.html, contact.html, reservations.html and so on.

If you want to show the logo on mobile devices, add the always-show-logo class to the wrapper.

<div id="wrapper" class="always-show-logo">
<!-- your content here -->
</div>
